# 📚 Lecturas Interactivas

Plataforma inmersiva de narrativa adaptativa que transforma la lectura clásica en una experiencia cinematográfica e interactiva.

## ✨ Características

- 🎭 **Narrativa Ramificada**: Decisiones que alteran el curso de historias clásicas
- 🎨 **Atmósfera Visual**: Efectos visuales temáticos (hojas, lluvia, humo, golondrinas...)
- 🎵 **Paisajes Sonoros**: Audio ambiental con sistema de eco fantasma
- 🤖 **IA Contextual**: Chat con Gemini para profundizar en la lectura
- ✨ **Experiencia Poética**: Modo karaoke y lectura silenciosa con efectos
- 📱 **PWA**: Instalable, offline-first, mobile-first

## 🚀 Desarrollo

```bash
npm install    # Instalar dependencias
npm run dev    # Servidor de desarrollo
npm run build  # Build de producción
```

## 🛠️ Stack

- **Build**: Vite 5
- **Frontend**: Vanilla HTML/CSS/JS (ES Modules)
- **IA**: Gemini 2.5 Flash (API + WebSocket Live)
- **Audio**: Web Audio API
- **PWA**: Manifest + Service Worker
